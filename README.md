This is a starter template for [Learn Next.js](https://nextjs.org/learn).

## Develop environment

- node : 15.0.1
- next : 9.5.3

### How to Node update

`$ brew update`

`$ brew upgrade nodebrew`

`$ nodebrew ls-remote`

`$ nodebrew install-binary v15.0.1`

`$ nodebrew list`

`$ nodebrew use v15.0.1`

`$ node -v`

=> v15.0.1
